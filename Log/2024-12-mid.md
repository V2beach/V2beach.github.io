# bfg

```sh
bfg --strip-blobs-bigger-than 1M
git reflog expire --expire=now --all && git gc --prune=now --aggressive
```

.git立刻從1.43G變成493M，反正.git版本控制數據不會上傳到github，就先這樣吧。

查看更改：
```sh
git diff
git show HEAD
git log --stat
```
